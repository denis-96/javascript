Client-side routing - роутинг на стороне клиента

React Router - внешняя библиотека для реализации роутинга 
в React приложениях

Функции React Router:
Роутинг,
Вложенные маршруты,
Параметры маршрутовб
Строки запросов,
Загрузчики данных,
Обработчики событий,
Перенаправление,
Страницы Not Found (404)
