Single Page Application - Одностраничное приложение

В одностраничных приложениях клиент запрашивает
только одну HTML страницу, JS и CSS файлы для всего приложения

Фреймворки и библиотеки для создания
одностраничных приложений:
React, Angular, Vue, Svelte, Ember, Ruby

Преймущества SPA (Single Page Application):
1. Веб страница не перезагружается при навигации по приложению
2. Моментальная перерисовка элементов веб страницы
3. Легче разворачивать, потому что SPA содержит только несколько статических файлов
4. Эффективное использование пропускной способности

Недостатки:
1. Для SEO (поисковой оптимизации) необходимо выполнить
   дополнительныю настройку сервера для корректного сканирования
   ботами, такими как GoogleBot
2. Большой размер JS файла влияет на время начальной загрузки приложения
