Фундаментальные принципы

React - библиотека для создания одностраничных приложений
SPA - single page application

Императивный подход:
говорит, как именно прийти к результату (конкретные шаги)
описывает конкретные шаги

Декларативный (используется в React):
говорит какой нужен результат
описывает итоговый результат (шаги не важны)

React основан на компонентах
React построен на модульной структуре - UI компонентах
User Interface components

Выучив один раз React можно создавать приложения для разных платформ


Внутренности React:
1. Реакт использует препроцессор JSX
2. Внутри реакта есть алгоритм, 
   который позволяет отслеживать какие компоненты изменились
   и перерисовывать только их
3. Технология Virtual DOM - лёгкая копия DOM
