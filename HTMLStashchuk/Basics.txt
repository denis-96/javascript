Как клиенты взаимодействуют с серверами

Когда вы открываете любой веб сайт,
клиент (веб браузер) скачивает небхоимые файлы с сервера

HTTP - HyperText Transfer Protocol
По этому протоколу клиенты устанавливают связь серверами
и сервера передают файлы клиенту

Протокол HTTP передаёт данные в окрытом виде (не зашифровывая)
Протокол TLS - Transport Layer Security

HTTP + TLS = HTTPS (S - secure - безопасный)
